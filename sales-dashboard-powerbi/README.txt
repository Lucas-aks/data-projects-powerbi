Dashboard de Gestão de Vendas


Descrição

Este projeto consiste em um dashboard desenvolvido no Power BI para análise e acompanhamento de vendas. O objetivo é oferecer uma visão clara e interativa sobre os principais indicadores de desempenho comercial.


Objetivos do Dashboard

Analisar o faturamento total em diferentes períodos.

Avaliar a performance de vendas por categoria, produto ou região.

Identificar tendências e apoiar a tomada de decisão baseada em dados.


Arquivos no Repositório

Gestao_de_Vendas.pbix → Arquivo principal do Power BI com o dashboard.

bd_vendas.xlsx → Base de dados utilizada para construção do dashboard.

screenshot_dashboard.png → Imagem ilustrativa do dashboard final.


Ferramentas Utilizadas

Microsoft Power BI

Microsoft Excel


Como Utilizar

Baixe os arquivos disponíveis no repositório.

Abra o arquivo Gestao_de_Vendas.pbix no Power BI Desktop.

Caso necessário, ajuste o caminho da base de dados (bd_vendas.xlsx) para que o Power BI encontre a fonte corretamente.


Resultados

O dashboard permite visualizar indicadores de faturamento e desempenho de vendas de forma intuitiva, auxiliando gestores na identificação de pontos fortes e oportunidades de melhoria.
