Dashboard de Energia – Power BI  

📊 Este projeto apresenta um dashboard de análise de consumo de energia.  
O objetivo é explorar padrões de consumo e visualizar métricas relevantes.  

🛠️ Ferramentas utilizadas
- Power BI
- Excel

📈 Insights principais
- Evolução do consumo ao longo do tempo
- Comparação de períodos
- Identificação de picos de energia

📂 Estrutura dos arquivos
- `Dash Energia.pbix` → Arquivo do Power BI
- `Base Energia.xlsx` → Base de dados usada no projeto
